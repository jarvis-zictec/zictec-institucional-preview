# ZICTEC — rascunho do novo site institucional

Rascunho navegável para avaliação interna, publicado via GitHub Pages sem interferir no WordPress atual.

## Versão v0.2 — multipágina

Esta etapa transforma o protótipo de uma página única em uma estrutura de site com navegação real, páginas separadas e submenu de soluções.

Páginas atuais:

- `index.html` — home institucional/comercial
- `quem-somos.html` — posicionamento e atuação da ZICTEC
- `solucoes.html` — hub de soluções
- `chamada-verificada.html` — Origem Verificada / STIR-SHAKEN
- `stfc-tools.html` — STFC Tools
- `sbc-prosbc.html` — SBC / ProSBC / Telcobridges
- `regulatorio-anatel.html` — Gestão regulatória Anatel
- `suporte.html` — suporte técnico especializado
- `segmentos.html` — segmentos atendidos
- `contato.html` — caminhos de contato/diagnóstico

Arquivos de apoio:

- `styles.css` — identidade visual compartilhada
- `build_site.py` — gerador simples do protótipo estático
- `assets/zictec-logo.png` — logo usado no header

## Escopo

- Arquitetura de informação v0.2
- Header com links reais e submenu de soluções
- Conteúdo inicial real derivado dos sites ZICTEC e da base comercial/telecom consolidada
- Padrão visual premium técnico para validação

Não contém publicação em domínio oficial, agendamento, campanhas ou alterações em WordPress.
